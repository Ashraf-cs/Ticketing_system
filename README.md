<h2>Technical support tickets system</h2>
<br><br>
<h3>About</h3>
<p>This system is intended to enterprises that have multiple branchs and each branch has a support technician or group of support technicains responsible for its support tickets.</p>
<p>A support technician can be responsible for one branch or more</p>
<br>
<h3>How it works</h3>
<ul>
  <li>A user initiate support ticket and send it to technical support team</li><br>
  <li>The system receives the ticket and look at technican or technicans group responsible for user branch tickets</li><br>
  <li>The system decide to which technicain will allocate the ticket based on the lowest one has tickets currently (at the time when the ticket received)</li><br>
  <li>The admin of the system can convert a ticket from a technicain to another even if the technicain is not responsible for it</li><br>
  <li>Tickets can be closed and reopened by the system admin</li><br>
</ul>
